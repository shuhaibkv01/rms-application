# RMS-Application
Rate Application Management Application
=================================================

### Steps for running application and test cases

##### Clone the project
    clone https://github.com/shuhaibkv01/rms-application.git
  
    
##### Build the package
    mvn package
##### Run the application
    mvn exec:java -Dexec.mainClass=com.xyz.rms.RmsApplication
##### Run the test cases
    mvn clean test
